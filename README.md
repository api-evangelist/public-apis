# Public APIs (public-apis)

Public APIs is a community-curated collective list of over 1,400 free APIs organized across 50+ categories, maintained by contributors and the team at APILayer. It is one of the most popular open-source projects on GitHub with over 400k stars. A companion REST API (davemachado/public-api) exposes the directory data programmatically over HTTPS with no authentication. This repository captures the APIs, developer tools, and machine-readable API artifacts for Public APIs.

## APIs

### Public APIs Directory

A collective list of free APIs organized by category including Animals, Finance, Weather, Geocoding, Government, Health, Machine Learning, Sports, and 40+ more categories.

- [Documentation](https://github.com/public-apis/public-apis#index)
- [GitHub Repository](https://github.com/public-apis/public-apis)

### Public APIs API

A RESTful API for programmatically accessing the public-apis directory data. Supports CORS and requires no authentication, with all responses served over HTTPS. Endpoints include `/entries`, `/random`, `/categories`, and `/health`.

- Base URL: `https://api.publicapis.org`
- [Documentation](https://github.com/davemachado/public-api)
- [GitHub Repository](https://github.com/davemachado/public-api)
- [OpenAPI Spec](https://raw.githubusercontent.com/api-evangelist/public-apis/refs/heads/main/openapi/public-apis-openapi.yml)
